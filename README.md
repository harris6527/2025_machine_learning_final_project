# 2025 Machine Learning Final Project — The Super-Pedagogue

這是專題的可評估封裝，包含核心原型程式與報告摘要。

## 內容
- `main.py`：診斷式導師原型，使用規則與可選 Sympy 進行錯誤偵測與引導。
- `report.md`：報告摘要版，含願景、資源分析、ML 架構、實作與討論。

## 執行方式
```bash
python main.py
```
- 輸入數學算式（例：`(a+b)^2 = a^2 + b^2`，`d/dx(x^2) = x`），程式會輸出啟發式提示，不直接給答案。
- 輸入 `exit` 或空行離開。

## 環境需求
- Python 3.8+
- 可選：`sympy`（若未安裝，程式自動以規則模式運作）

## 檔案說明
- `main.py`：含認知追蹤模擬（連續同一錯誤時升級提示）、教學圖譜規則、與簡易 REPL。
- `report.md`：包含雙軌知識庫（教學圖譜 + 黃金教材庫）、語法 vs 語意驗證討論、以及模擬對話範例。
